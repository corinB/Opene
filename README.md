## 소개
<h4>옷 사진을 통해 비슷한 색상의 원하는 스타일을 추천해주는 플랫폼</h4>




## 기능
- 상하의 구분 및 색상 추출 기능
- 서버 내에 저장 된 사진들 중 비슷한 색을 가진 코디 사진 선정
- 스타일 추천




## 설치
- flask 서버 작동시 conda 환경 필요
- python 3.10 환경
- packagelist 인스톨 필요
  ```bash
  conda install -r packagelist.txt
